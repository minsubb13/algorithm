# queue

```#include <queue>```

- 선언

    FIFO (First In First Out) 구조

    ```queue<int> q;```

- 주요 메서드

    ```push(element)```

    ```pop();``` 
    : 큐의 맨 앞 요소 제거 (반환x)

    ```front();```
    : 큐의 맨 앞 요소 반환

    ```back();```
    : 큐의 맨 뒤 요소 반환

    ```empty();```
    return true or false

    ```size();```
